# goexpress
> 一个简单的类似与node.js express 框架


## 功能点

1. 一个独立的服务
2. 可以访问静态页面
3. 可以提供接口类似请求
4. 添加缓存 + redis 缓存
5. http请求接口请求
6. 处理页面请求
7. 路由
8. 数据库操作
